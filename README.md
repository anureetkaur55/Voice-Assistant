# Python Voice Assistant 🎙️

## Overview

This project is a Python-based Voice Assistant that can listen to voice commands, respond using text-to-speech, answer questions using AI models, search Wikipedia, play songs on YouTube, open websites, tell the current time, and send WhatsApp messages.

The assistant uses Speech Recognition for voice input and Text-to-Speech technology to provide spoken responses, creating a hands-free interactive experience.

## Features

* 🎤 Voice command recognition
* 🔊 Text-to-speech responses
* 🤖 AI-powered question answering
* 📚 Wikipedia search and summaries
* 🎵 Play songs on YouTube
* 🌐 Open websites like Google and YouTube
* ⏰ Tell the current time
* 💬 Send WhatsApp messages
* 🛑 Voice-controlled exit command

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* OpenAI API
* Hugging Face API
* Wikipedia API
* PyWhatKit
* Requests

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/voice-assistant.git
```

2. Navigate to the project directory

```bash
cd voice-assistant
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

## Usage

1. Add your API keys in the script.
2. Connect a microphone to your system.
3. Run the program:

```bash
python main.py
```

4. Speak commands such as:

* "What is Artificial Intelligence?"
* "Open YouTube"
* "Open Google"
* "Play Shape of You"
* "Tell me the time"
* "Search Wikipedia for Python"
* "Send message"
